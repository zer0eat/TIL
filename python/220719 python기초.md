# Python

## 변수(Variable) = 식별자

- 데이터를 담는 상자 (한 개의 데이터를 저장)

- 추상화 : 내부의 복잡한 것을 숨기고 필요한 것만 드러냄

- 변수의 사용 이유
  
  - 코드 가독성 증가
  
  - 의미 단위로 작성 가능
  
  - 코드 수정 용이

- 변수의 할당 
  
  - 연산자(=)를 통해 값을 할당(assignment)

- 변수 이름 규칙
  
  - 영문 알파벳, 언더스코어(_), 숫자만 가능
  
  - 숫자는 맨앞에 나올 수 없음
  
  - 길이 제한이 없고, 대소문자 구별함
  
  - 예약어는 사용할 수 없음
    
    ['False', 'None', 'True', '\__peg_parser__', 'and', 'as', 'assert', 'async', 'await', 'break', 'class', 'continue', 'def', 'del', 'elif', 'else', 'except', 'finally', 'for', 'from', 'global', 'if', 'import', 'in', 'is', 'lambda', 'nonlocal', 'not', 'or', 'pass', 'raise', 'return', 'try', 'while', 'with', 'yield']
  
  - 내장함수 모듈 등의 이름도 사용할 수 없음(동작 불가)

- 산술연산자(Arithmetic Operator)
  
  | 덧셈  | 뺼셈  | 곱셈  | 나눗셈 | 몫   | 거듭제곱 |
  |:---:|:---:|:---:|:---:|:---:|:----:|
  | +   | -   | *   | /   | //  | **   |
  
  - 사칙연산과 같은 우선 순위

## 자료형(Datatype)

1. Boolean Type(논리자료형/불린형)
   
   - 참과 거짓을 표현(True/False)
   
   - 비교, 논리 연산에서 활용
   
   - 비교연산자    
   
   | <   | <=  | >   | >=  | ==  | !=   | is          | is not    |
   |:---:|:---:|:---:|:---:|:---:|:----:|:-----------:| --------- |
   | 미만  | 이하  | 초과  | 이상  | 같음  | 같지않음 | 객체 아이덴티티OPP | OPP가 아닌경우 |
   
   - 논리연산자
     
     - A and B : A와 B 모두 True시, True
     
     - A or B : A와 B 모두 False시, False
     
     - Not : True>>False / False>>True
       
       - 우선순위 Not >> or >> and
     
     - 주의할 점
       
       - Falsy : Flase는 아니지만 False로 취급되는 값
         
         - 0, 0.0, (), {}, [], None, ''
       
       - 단축평가
         
         - and연산  : 앞이 False면 첫번째 값 반환
         
         - or연산 : 앞이 True면 첫번째 값 반환

2. Numeric Type(수치형)
   
   - Int(정수)
     
     - 수학연산 가능 / 진수로 표현 가능(0b, 0o, 0x)
   
   - Float(실수)
     
     - 유리수와 무리수 
     
     - 부동 소수점 때문에 연산결과가 다를 수 있음
       
       (2진법으로 계산하기 때문에)
   
   - Complex(복소수)

3. String Type(문자형)
   
   - str 
     
     - 따옴표를 활용하여 표기
     
     - 문자열을 묶을 때 동일한 문장부호 활용 / PEP8 하나의 문장부호  통일 권장
     
     - 중첩 따옴표
       
       작은 따옴표가 들어 있는 경우, 큰따옴표로 문자열 생성
       
       큰 따옴표가 들어 있는 경우, 작은따옴표로 문자열 생성
     
     - 삼중따옴표
       
       따옴표를 삼중으로 사용 / 여러줄을 나눠 입력할 때 사용
     
     -  escape sequence
   
   | \n  | \t  | \r     | \0      | \\\ | \'     | \"     |
   |:---:|:---:|:------:|:-------:|:---:|:------:|:------:|
   | 줄바꿈 | 탭   | 캐리지 리턴 | 널(Null) | \   | 단일인용부호 | 이중인용부호 |
   
   - 문자열 연산
     
     - 덧셈 : 문자열 연결
     
     - 곱셈 : 문자열 반복
     
     - string interpolation
       
       - %-formatting `ex)name=kim, print('%s', %name)`
       
       - str.format() `ex)name=kim, print('{}'.format(name))`
       
       - f.string`ex)name=kim, print(f'{name}) `
   4. None
      
      - 값이 없음을 표현하기 위한 타입
      
      - 반환 값이 없는 함수에서 사용

## 컨테이너

- 정의 : 여러개의 값(데이터)을 담을 수 있는 것(객체), 서로 다른 자료형을 저장할 수 있음

- 분류 : Ordered(순서가 있는 데이터) vs Unordered(순서가 없는 데이터)

- 시퀀스형 / Ordered(순서가 있는 데이터)
  
  1. 리스트(list) : 여러개의 값을 순서가 있는 구조로 저장
     
     - 특징 : 가변 자료형, 리스트 안에 리스트를 넣을 수 있음
     
     - 생성 : 리스트명 = [] / 리스트명 = list()
     
     - 접근 : 리스트명[i]
     
     - 변경 : 리스트명[i] = '변경할 값'
     
     - 추가 : list()+['추가할 값']
  
  2. 튜플(tuple) : 여러 개의 값을 순서가 있는 구조로 저장
     
     - 특징 : 불변 자료형
     
     - 생성 : 튜플명 = () / 튜플명 = tuple()
     
     - 접근 : 튜플명[i]
     
     - 튜플대입 : 우변의 값을 좌변에 할당하는 과정
     
     - 주의사항
       
       - 단일항목 튜플 생성 시 
         
         - 마지막 값 뒤에 쉼표를 붙여야 함(안붙이면 int로 저장)
       
       - 복수항목 튜플 생성 시 
         
         - 마지막 값 뒤에 쉼표를 안붙여도 되나 붙이는 것을 권장
  
  3. 레인지(range) : 숫자의 시퀀스를 나타내기 위해 사용 / 반복문과 함께 사용
     
     - 기본형 : range(n)
       
       - 0부터 n-1까지 숫자 시퀀스
     
     - 범위 지정 : range(n,m)
       
       - n부터 m-1까지 숫자 시퀀스
     
     - 범위 및 스텝 지정 : range(n,m,s)
       
       - n부터 m-1까지 s만큼 증가시키는 숫자 시퀀스
       
       - s에 -1 대입 시 역순
     
     - 슬라이싱 연산자
       
       - 인덱스와 콜론을 사용하여 특정부분 잘라냄
       
       - [n:m]일 때 n부터 m-1까지 포함
       
       - [n\:m:s]일 때 n부터 m-1까지 s간격으로 포함
       
       - [::] >> 전체
       
       - [::-1] >> 전체를 역순으로

- 비시퀀스형 / Unordered(순서가 없는 데이터)
  
  1. 셋(set) : 집합 / 중복되는 요소 없이 순서에 상관 없는 데이터의 묶음
     
     - 특징 
       
       - 가변 자료형 / 집합연산 가능
       
       - 데이터 중복을 허용하지 않아 중복되는 원소는 하나만 저장
       
       - 순서가 없어 인덱스로 접근 불가
     
     - 생성 : 셋명 = {} / 셋명 = set()  (빈 셋은 set()으로만 만들 수 있음)
     
     - 셋 연산자
       
       | |   | &   | -   | ^                        | 여집합은 |
       |:---:|:---:|:---:|:------------------------:|:----:|
       | 합집합 | 교집합 | 차집합 | 대칭차집합 (두 집합의 교집합을 뺀 나머지) | 없음   |
  
  2. 딕셔너리 : key-value 쌍으로 이뤄진 자료형 (unordered지만 3.7버전부터 ordered)
     
     - key : 변경 불가능한 데이터(immutable)만 활용가능
       
       - string, integer, float, boolean, tuple, range
     
     - value : 데이터 형태에 관계 없이 활용가능
     
     - 생성 : 딕셔너리명 = {'a' : 1} / 딕셔너리명 = dict(a = 1)

- 형변환 : 데이터 형태의 변환
  
  - 암시적 형 변환(implicit)
    
    - 사용자가 의도하지 않고, 파이썬 내부적으로 자료형을 변환하는 경우
    
    - boolean type + numeric type 연산 시
  
  - 명시적 형 변환(Explicit)
    
    - 사용자가 특정 함수를 활용하여 의도적으로 자료형을 변환하는 경우
  
  - 컨테이너 형변환
    
    |            | string | list    | tuple   | range | set     | dictionary |
    |:----------:|:------:|:-------:|:-------:|:-----:|:-------:|:----------:|
    | string     | \      | O       | O       | X     | O       | X          |
    | list       | O      | \       | O       | X     | O       | X          |
    | tuple      | O      | O       | \       | X     | O       | X          |
    | range      | O      | O       | O       | \     | O       | X          |
    | set        | O      | O       | O       | X     | \       | X          |
    | dictionary | O      | O(key만) | O(key만) | X     | O(key만) | \          |
